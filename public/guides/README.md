# Wear Me PDF guides for integration emails

These files are attached when sending the Wear Me integration email from admin (per platform)
or from the retailer dashboard (**`wear-me-custom.pdf`**).

| Filename | Used for |
| --- | --- |
| `wear-me-shopify.pdf` | Shopify installs |
| `wear-me-wordpress.pdf` | WordPress installs |
| `wear-me-wix.pdf` | Wix installs |
| `wear-me-squarespace.pdf` | Squarespace installs |
| `wear-me-custom.pdf` | Custom / retailer “email developer” flow |

Ship real platform-specific PDFs here — the repo currently uses the same placeholder sample for each file so deliveries work end-to-end in development.
